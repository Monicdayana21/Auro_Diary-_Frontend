# Auro Diary - Frontend ✨🌸

The aesthetic frontend for Aura Diary, built with React and Vite.

## Deployment
This project is configured for easy deployment on **Vercel**. 

## Tech Stack
- React
- Vite
- Axios
- Chart.js
- Framer Motion

## Getting Started
1. `npm install`
2. `npm run dev`
